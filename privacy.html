Privacy Policy for Jarak Bot

Last Updated: July 15, 2026

1. Data We Collect

Jarak Bot may collect and process limited information automatically provided by the Discord API to function correctly. This data may include:

  • Discord User IDs: To identify users for features like moderation, tracking economy systems, or user preferences.
  • Server (Guild) IDs & Channel IDs: To save server-specific configurations, custom prefixes, and settings.
  • Message Content: Only processed temporarily to read and execute commands initiated by users. Message content is not permanently stored or logged unless required for a specific logging feature you explicitly enable.

2. How We Use Data

The data collected is used solely to operate, maintain, and provide the core features of Jarak Bot. We do not sell, trade, or rent user data to third parties under any circumstances.

3. Data Retention

We only store data for as long as necessary to provide the bot's services. Server-specific data is kept while the bot is a member of that server. If Jarak Bot is removed (kicked/banned) from a server, data related to that server may be queued for deletion.

4. Data Deletion Requests

Users have the right to request the total deletion of any stored data associated with their Discord account or server. To request data deletion, please contact the development team through our official channels.

5. Security

We implement basic, industry-standard security measures to protect stored information. However, no method of transmission over the internet or electronic storage is 100% secure, and we cannot guarantee absolute data security.

6. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We encourage users to check this page periodically for any changes. Continued use of the bot implies your agreement to the current policy.

7. Contact Us

If you have any questions about this Privacy Policy, your data, or wish to request data removal, please reach out via our support channels.
